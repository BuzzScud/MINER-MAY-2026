5. Console Errors

Check your browser console (F12) for any red errors.

Common errors to look for:

  • 404 Not Found errors for CSS/JS files
  • CORS errors
  • JavaScript syntax errors
  • WebSocket connection failures

6. Network Tab Check

Open DevTools (F12) → Network tab and look for:

  • /static/css/output.css - Should be 200 OK
  • /static/js/preline.js - Should be 200 OK
  • /static/js/dashboard.js - Should be 200 OK
  • /socket.io/ - WebSocket connection
